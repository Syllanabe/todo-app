# TaskFlow - Mini Todo App

## Description

TaskFlow est une mini-application de gestion de tâches développée avec HTML, CSS et JavaScript.

L'application permet à l'utilisateur d'ajouter des tâches, de les marquer comme terminées, de les supprimer et de filtrer les tâches selon leur état.

Le projet a été réalisé dans le cadre d'un exercice visant à créer et déployer une mini-application en utilisant des outils d'intelligence artificielle.

## Fonctionnalités

- Ajouter une nouvelle tâche
- Marquer une tâche comme terminée
- Supprimer une tâche
- Filtrer les tâches :
  - Toutes
  - En cours
  - Terminées
- Supprimer toutes les tâches terminées
- Sauvegarder les tâches avec `localStorage`
- Basculer entre le mode clair et le mode sombre
- Interface responsive adaptée aux écrans mobiles

## Technologies utilisées

- HTML
- CSS
- JavaScript
- GitHub Pages
