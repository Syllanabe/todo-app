# Spécification du projet - TaskFlow

## Nom du projet

TaskFlow - Mini Todo App

## Objectif

L'objectif du projet est de créer une mini-application web permettant à un utilisateur de gérer ses tâches quotidiennes de manière simple et intuitive.

## Type d'application

Application Todo.

## Utilisateur cible

Toute personne souhaitant organiser ses tâches personnelles, scolaires ou professionnelles.

## Fonctionnalités principales

### 1. Ajouter une tâche

L'utilisateur peut saisir une tâche dans un champ texte et cliquer sur le bouton "Ajouter".

### 2. Afficher la liste des tâches

Toutes les tâches ajoutées sont affichées dans une liste.

### 3. Marquer une tâche comme terminée

L'utilisateur peut cocher une tâche pour la considérer comme terminée.

### 4. Supprimer une tâche

Chaque tâche dispose d'un bouton permettant de la supprimer.

### 5. Filtrer les tâches

L'utilisateur peut afficher :
- toutes les tâches ;
- les tâches en cours ;
- les tâches terminées.

### 6. Supprimer les tâches terminées

Un bouton permet de supprimer toutes les tâches déjà terminées.

### 7. Sauvegarde locale

Les tâches sont sauvegardées dans le navigateur grâce à localStorage.

### 8. Mode clair / sombre

L'utilisateur peut changer le thème visuel de l'application.

## Technologies utilisées

- HTML
- CSS
- JavaScript
- GitHub Pages

## Contraintes

- Aucun framework utilisé
- Application responsive
- Code simple et lisible
- Données sauvegardées localement dans le navigateur

## Critères de réussite

Le projet est réussi si :
- l'utilisateur peut ajouter une tâche ;
- l'utilisateur peut supprimer une tâche ;
- l'utilisateur peut marquer une tâche comme terminée ;
- les tâches restent sauvegardées après actualisation de la page ;
- l'application est accessible via GitHub Pages.
